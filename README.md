# LyceeApp

Application mobile React Native pour consulter les informations sur les lycées d'Île-de-France.

## Installation

1. Assurez-vous d'avoir Node.js installé sur votre machine.
2. Ouvrez un terminal à la racine du projet.
3. Installez les dépendances avec la commande suivante :

   npm install

## Utilisation

Pour lancer l'application en mode développement :

   npx expo start

Une fois la commande lancée :
- Scannez le QR code affiché dans le terminal avec l'application Expo Go (disponible sur Android et iOS).
- Ou appuyez sur 'a' pour lancer sur un émulateur Android.
- Ou appuyez sur 'i' pour lancer sur un simulateur iOS.
- Ou appuyez sur 'w' pour lancer dans un navigateur web.

## Fonctionnalités principales

- Liste des lycées avec recherche
- Filtrage par académie (Paris, Créteil, Versailles)
- Filtrage par type d'établissement (Public, Privé)
- Accès aux contacts (email) et localisations
